# Moist Dads Website

- Make sure your in the root directory (moistdads folder)

## General

### Prerequisites
1. install mongodb
2. install ffmpeg
2. install graphicsmagick

### Install
1. npm run i-server
2. npm run i-client

### Local Run
1. npm run server 
2. 📟 (new terminal)
3. npm run client
